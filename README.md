# Pearland Coffee Cart

Modern Next.js landing page for Pearland Coffee Cart.

## Local Preview

```bash
npm install
npm run dev
```

## Deploy on Netlify

1. Push this project to a GitHub repository.
2. In Netlify, choose **Add new site** then **Import an existing project**.
3. Connect the repository.
4. Use these settings:
   - Build command: `npm run build`
   - Publish directory: `.next`
5. Deploy the site.

The project includes `netlify.toml`, so Netlify should detect the build settings automatically. Netlify supports modern Next.js projects through its OpenNext adapter.
